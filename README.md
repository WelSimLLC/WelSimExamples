# Examples
WelSim Tests and Examples

!important
This repository is obsolete. If you want to obtain WELSIM examples, please run the automated regression tests and save the test project files to get the examples. The tests are located at [https://github.com/WelSimLLC/WelSimAutoTests](https://github.com/WelSimLLC/WelSimAutoTests).

### Usage
Open the *.wsdb project file using WELSIM application. 

If you need a license file, please refer the activation process [https://welsim.com/activation](https://welsim.com/activation), or contact info@welsim.com.
