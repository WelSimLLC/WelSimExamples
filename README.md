# Examples
Tests and Examples for WELSIM

> [!IMPORTANT]
> This repository is obsolete.

If you want to obtain WELSIM examples, please run the automated regression tests and save the test project files to get the examples. The tests are located at [https://github.com/WelSimLLC/WelSimAutoTests](https://github.com/WelSimLLC/WelSimAutoTests).

### Usage
Simply drag the *.wsdb file and drop to the WELSIM main window.

If you need a license file, please refer the activation process [https://welsim.com/activation](https://welsim.com/activation), or contact info@welsim.com.
